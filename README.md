## 📌 Overview

**TravelZone** is a front-end travel website designed to provide users with an immersive trip planning experience. The site showcases popular tour types, destination highlights, cultural content, and an email newsletter section — all wrapped in a clean, modern UI built entirely with vanilla web technologies.

This project demonstrates proficiency in:
- Semantic HTML5 structure
- Responsive CSS layout and styling
- DOM manipulation with JavaScript
- UI/UX design principles for web applications

## ✨ Features

- 🧭 **Responsive Navigation Bar** — with mobile hamburger menu toggle
- 🏝️ **Hero Section** — full-screen banner with a trip booking call-to-action
- 🎿 **Tour Categories** — Ski, Excursion, Shopping, SPA, and Wedding tours
- 🏕️ **Holiday Spotlight** — featured destination with description (East Nusa Tenggara / Komodo Island)
- 🌋 **Popular Tours** — card-based layout showcasing destinations like Mount Bromo, East Java
- 🎎 **Indonesian Culture** — dedicated culture section with imagery
- 📧 **Newsletter Signup** — email subscription form with CTA
- 🔗 **Social Media Links** — Instagram, Twitter, Facebook integrations
- 🦶 **Footer** — organized with Services, About, Help, and Social columns

## 🚀 Getting Started

No build tools or dependencies required. Just clone and open in a browser.

**1. Clone the repository:**

```bash
git clone https://github.com/MOHAMMADSHADULLA/travelzone.git
cd travelzone
```

**2. Open in browser:**

```bash
open index.html
```

Or right-click `index.html` → *Open with* → your preferred browser.

> The project uses CDN-hosted libraries (Boxicons, RemixIcon, Google Fonts), so an internet connection is needed for icons and fonts to load correctly.
